** Usando Emacs para tener Ayuda sobre Emacs
**Comandos Fáciles**
1. **Danilo Unapucha** C-n. Move cursor vertically down ARG lines.
2. **Sebastian Arevalo** C-x C-s Save currend buffer
3. **Michael Sornoza** C-a. Runs the command org-beginning-of-line (found in org-mode-map).
4. **Matias Palomeque** M-f. Move point forward ARG words (backward if ARG is negative).

**Comandos Dificiles**
1. **Danilo Unapucha** C-v. Scroll text of selected window upward ARG lines; or near full screen if no ARG.
2. **Sebastian Arevalo** C-k kill or remove de line where the cursor is at.
3. **Michael Sornoza** C-u Runs the command universal-argument (found in global-map), which is an interactive native-compiled Lisp function in ‘simple.el’.
4. **Matias Palomeque** C-/. Undo some previous changes.
